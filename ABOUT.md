# Doctor Who Newsfeed

This repo exists because the
[BBC's Doctor Who website](https://doctorwho.tv/news-and-features) stopped
publishing a web feed a while ago and I wanted a feed to plug into
[Planet Doctor Who](https://drwho.theplanetarium.org/).

If you're interested:

* The code - [https://github.com/davorg/drwho-newsfeed](https://github.com/davorg/drwho-newsfeed)
* The web feed - [https://davorg.dev/drwho-newsfeed/doctorwho.xml](https://davorg.dev/drwho-newsfeed/doctorwho.xml)
